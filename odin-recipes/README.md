# Odin Recipes – The Odin Project

## 📌 Overview
This project is part of **The Odin Project** curriculum.  
It is a simple HTML-based website that showcases multiple recipes using basic HTML elements such as headings, paragraphs, lists, and emphasis tags.

The goal of this project is to practice structuring content on a webpage and understanding how HTML works.

## Features
- Displays multiple recipes:
  - Chocolate Chip Muffins 🧁
  - Pizza 🍕
  - Maple Coffee Cake 🍰
- Uses semantic HTML elements
- Includes:
  - Descriptions
  - Unordered lists (ingredients)
  - Ordered lists (steps)
- Clean and readable structure

## Technologies Used
- HTML5

## What I Learned
- How to structure a webpage using HTML
- Difference between ordered (`<ol>`) and unordered (`<ul>`) lists
- Use of semantic tags like `<h1>`, `<p>`, `<em>`, and `<strong>`
- Organizing files into folders
- Writing clean and readable code

## ▶ How to Run
1. Download or clone this repository  
2. Open the folder  
3. Double-click on `index.html`  
4. View the website in your browser
